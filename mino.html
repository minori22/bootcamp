Particulas

1) S は S です


  • わたし は ダニーです。
  • しごと は プログラマーです。
  • くに は エクアドルです。
  • これ は かさです。


2) S が、、、


  • ねこ が すきです。
  • たなかさん が います。
  • じどうしゃ が あります。


3) Lugar に / へ いきます、きます、かえります


  • あした、東京　へ いきます。
  • きのう、みのり と 日本 へ きました。
  • マリオさんは うち へ かえりました。


3) Hora に V


  • あした、７じ に べんきょうします。
  • けさ、９じ に おきました。
  • ミラーさん は まいばん 9:30 に ねます。
  • あした、7じ に いきます/ きます/ かえります。


3) Transporte / lugar / medio で V


  • おはし で たべます。
  • あおい ペン で かきます。
  • タクシー で いきます。
  • まいにち、じむしょ で はたらきます。


4) Persona / animal と V


  • らいしゅう、かぞく と フランス　へ　いきます。
  • サントスさん は ともだち と かえりました。
  • レストラン で はは と たべました。


5) S の S


  • これは わたし の うでどけい です。
  • あれ は わたし の てちょう じゃありません。
  • さとうさん の いえ は きれいです。


6) S を V


  • おととい、ラーメン を たべました。
  • みのり に パソコン を かしました。
  • かようび と もくようび は プログラミング を おしえます。


7) S も S/ V


  • ワイン もすきです。
  • いぬ も ねこ も すきです。
  • ジョンさん は アメリカ人です。 メアリーさん も アメリカ人です。


8) S や S が あります / います / etc.

  • えんぴつ や ノート が あります。
  • かいしゃ に リーさん や トムさんがいます。
  • あそこ に びょういん や ぎんこう が あります。
